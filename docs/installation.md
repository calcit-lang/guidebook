# Installation

You need to install Rust first, then install Calcit with Rust:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

then Calcit is available as a Rust command line:

```bash
cr eval "echo |done"

cargo install calcit
```
